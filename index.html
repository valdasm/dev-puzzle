Frequently Asked Questions

What can I find here?

A list of Azure and AWS services (more providers to come), its alternatives, available integrations and limitations. The website will focus most on available and missing service-to-service integrations - “can a service X work with service Y”

Who can contribute?

Everyone. It is a public GitHub repository where everyone can contribute. In addition, if there are any inaccuracies, please contact us, or leave a comment. It is a work in progress.

Is it a commercial product?

No. The main purpose of this website is publicly available knowledge sharing.

Why did you launch it?

I am an IT architect, and the idea to build such repository came when I was preparing for Azure architect certification. I know how painful it is to discover tiny limitations that discard an architecture. It is my attempt to create a holistic view of all available cloud services.
